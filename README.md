## 🌿 Структура веток репозитория

В данном проекте согласно требованиям КИС организована работа со следующими ветками:

1. **`main`** — основная производственная ветка репозитория. Содержит финальную стабильную версию проекта, готовый отчет и общую документацию.
2. **`examples`** — учебная ветка (Задание 4). Содержит разработанные тренировочные примеры программ на языке C для изучения базового синтаксиса SQLite C API и задокументированный список примеров в файле `EXAMPLES.md`.
3. **`dev`** — ветка разработки (Задание 5). В этой ветке велась поэтапная реализация основного консольного Си-приложения для управления сущностью «Школьник» с поддержкой транзакций и BLOB-данных. Все изменения перенесены в `main` через Pull Request.
