# Консольное Си-приложение для работы с БД «Школьник» (Задание 5)

Данный подпроект представляет собой консольное управляющее приложение на языке Си, разработанное в соответствии с моделью КИС для работы с реляционной СУБД SQLite3.

## 🚀 Реализованный функционал
* **Интерактивное консольное меню** для выбора административных действий.
* **Параметризованные SQL-запросы** (`SELECT`, `INSERT`, `DELETE`) для предотвращения SQL-инъекций и корректного поиска.
* **Поиск по параметрам:** выборка учеников по конкретному классу (общему полю для нескольких строк).
* **Работа с бинарными данными (BLOB):** функции для загрузки локального файла изображения в базу данных и последующего экспорта фотографии конкретного школьника из БД обратно в файл на диске.
* **Режимы записи данных:** наглядная демонстрация разницы работы СУБД в пошаговом режиме (`AUTOCOMMIT`) и пакетном режиме с использованием явного управления транзакциями (`BEGIN TRANSACTION` / `COMMIT` / `ROLLBACK`).

## 📁 Содержимое каталога
* `main.c` — исходный код программы на языке C.
* `Makefile` — скрипт автоматизированной компиляции и линковки с флагом `-lsqlite3`.
* `school_app` — исполняемый файл приложения (появляется после сборки).
