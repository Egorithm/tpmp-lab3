# Изученные примеры SQLite C API (по материалам сайта ZetCode)

В данной директории размещены тренировочные примеры кода Курдеко Егора (13 группа).

## Список файлов:

1. **`example1_gr13_KurdekoEgor.c`**
   - **Описание:** Получение метаданных о СУБД с помощью функции `sqlite3_libversion()`.

2. **`example2_gr13_KurdekoEgor.c`**
   - **Описание:** Работа со структурой `callback`. Создание таблиц и вывод через `sqlite3_exec()`.

3. **`example3_gr13_KurdekoEgor.c`**
   - **Описание:** Демонстрация атомарности ручных транзакций (`BEGIN TRANSACTION` / `COMMIT`).
